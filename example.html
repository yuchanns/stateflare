/

Usage

Add the following code to your website:

<script defer src="https://your-worker.workers.dev/track.js"></script>
<label id="uv_label">UV</label> /
<label id="pv_label">PV</label>

Description

  • PV (Page Views): Total number of page visits to your site (increments on each visit)
  • UV (Unique Visitors): Number of unique visitors based on IP + User-Agent hash (permanent deduplication)

Features

  • Automatic site tracking based on referrer URL
  • Multi-site support with separate counters for each origin
  • Persistent storage using Cloudflare D1
  • Privacy-focused: uses hashed IP + User-Agent for identification
  • No cookies or localStorage required
